Как открыть личное дело?
Выберите в главном меню Open Personal 
далее, выберите в октрывемся окне Open, затем идите по этому пути.
место-где-расположена-ваша-программа\Personal\Имя-личного-дела\Main.dbf
Откроется файл в текстовом поле, вы можете редактировать его и использовать 
tools
--english 
How do I open a personal file?
Select Open Personal in the main menu
next, select Open in the window that opens, then follow this path.
place-where-your-program-is-located\Personal\Personal-case-name\Main.dbf
The file will open in a text field, you can edit it and use 
tools
